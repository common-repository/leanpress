LeanPub on WordPress. Import your LeanPub books and view them on your WordPress site. List your coupons and edit them within the admin area. 

== Description ==

LeanPress is a WordPress plugin that will bring your LeanPub books into your WordPress site. You can view them even in the public area of the site. Images of your eBooks will be downloaded and imported into WordPress media so you can easily use them to show your eBook in other WordPress places.

Using this plugin you can list your coupons for a desired eBook and even edit them within the WordPress area. For each coupon in the edit form there is a direct link to that coupon.

Want to become a self publisher? Visit also [www.leanpub.com](http://www.leanpub.com) and see how this plugin could be of any use to you!


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the folder 'leanpress' to the `/wp-content/plugins/` directory or install them through WordPress Plugin area within the Admin section
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Insert your LeanPub API key to start listing coupons and importing eBooks.

== Frequently Asked Questions ==

= Where can I find the LeanPub API key? =

Your LeanPub API key can be found [here](https://leanpub.com/author_dashboard/settings) (You must be signed in to get it).

= Can I import others' books also? =

Yes you can. LeanPub API enables you to import even books there are not yours. The only thing that you will not get if they are not your own is the direct link to downloads of PDF, EPUB and MOBI files and some other more detailed data about the book.
